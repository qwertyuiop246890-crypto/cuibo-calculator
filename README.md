# 日圓 → 台幣 代購計算機（進階版）

這是專為日本代購、自用記帳設計的進階版計算機，支援：

- 匯率（TWD / 100 JPY）
- 國外交易手續費 %
- 代購費 %（視為成本）
- 額外利潤率 %
- 國際運費（重量 × 每克運費）
- 刷卡手續費 %
- 營業稅 %

自動計算：

- 商品成本（台幣）、國外手續費、台幣成本
- 代購費（成本）、預估運費、刷卡手續費、營業稅
- **最終總成本（含代購＋運費＋刷卡＋稅）**
- 含利潤售價（未稅未含運）、含運售價（未稅）、刷卡後金額（未稅）、最終售價（含稅）
- **實際利潤與實際利潤率（相對總成本）**

## 使用方式

1. 將整個專案資料夾上傳到 GitHub（或任何靜態空間），例如 `jp2twd_pwa_mobile_v2`。
2. 啟用 GitHub Pages（使用 `main` / `master` 分支的 `root` 或 `docs` 目錄）。
3. 用手機打開網址即可使用。可以「加入主畫面」變成 PWA App。

> 匯率欄位使用「TWD / 100 JPY」，例如匯率 0.225 則輸入 22.5。
